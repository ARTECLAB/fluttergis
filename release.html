Hack
split-per-abi:
de 25MB a 9MB
Un flag que reduce el APK un 60% — obligatorio para produccion
Hack bueno de la clase
Compilar el APK optimo + obfuscacion
Comandos de build para produccion
# APK universal (EVITAR en produccion) flutter build apk --release # Resultado: ~25MB # APK por arquitectura (RECOMENDADO) flutter build apk --release --split-per-abi # Resultado: # app-arm64-v8a-release.apk ~10MB # app-armeabi-v7a-release.apk ~9MB # app-x86_64-release.apk ~9MB # Con obfuscacion (proteger logica) flutter build apk --release --split-per-abi --obfuscate --split-debug-info=./debug_symbols # App Bundle para Play Store flutter build appbundle --release # Play Store genera APKs optimizados # por dispositivo automaticamente
APK universal~25MB — incluye todas las arquitecturas
APK arm64~10MB — solo para 64-bit (mayoria)
APK arm32~9MB — solo para 32-bit (dispositivos viejos)
Ahorro60% menos — cada usuario descarga lo que necesita
⚠️
--obfuscate SOLO afecta el binario compilado — el codigo fuente .dart permanece intacto. Guardar los debug_symbols para leer stack traces de crashes.
