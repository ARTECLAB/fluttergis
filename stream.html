Concepto clave
Future vs Stream — la diferencia que lo cambia todo
📸
Future — una foto
Devuelve un solo valor y termina. getCurrentPosition() da la posición actual y no sigue actualizando.

Úsalo para: posición inicial al abrir la app.
🎥
Stream — una película
Emite valores continuamente mientras esté activo. getPositionStream() emite una posición nueva cada vez que te mueves.

Úsalo para: seguimiento GPS en tiempo real.
Los dos métodos de geolocator
// Future: posición actual (una vez) final pos = await Geolocator.getCurrentPosition(); // Stream: posición continua final stream = Geolocator.getPositionStream( locationSettings: LocationSettings( accuracy: LocationAccuracy.high, // Hack: solo actualiza cada 5m distanceFilter: 5, ), ); // Escuchar el stream stream.listen((Position pos) { final latLng = LatLng( pos.latitude, pos.longitude); // actualizar mapa aquí });

class _MapaState extends State<MapaScreen> with WidgetsBindingObserver { StreamSubscription? _gpsSub; @override void initState() { super.initState(); WidgetsBinding.instance .addObserver(this); _iniciarGPS(); } void _iniciarGPS() { _gpsSub = Geolocator .getPositionStream( locationSettings: LocationSettings( distanceFilter: 5, )) .listen(_onNuevaPosicion); } @override void didChangeAppLifecycleState( AppLifecycleState state) { if (state == AppLifecycleState.paused) { _gpsSub?.pause(); // App al fondo } else if (state == AppLifecycleState.resumed) { _gpsSub?.resume(); // App activa } } @override void dispose() { _gpsSub?.cancel(); WidgetsBinding.instance .removeObserver(this); super.dispose(); } }

✓
WidgetsBindingObserver
Detecta cuando la app pasa al background (paused) y pausa el stream GPS. Cuando el usuario vuelve (resumed) lo reactiva. Sin código nativo.

StreamBuilder + PolylineLayer
Mapa que sigue al usuario
final _trayectoria = <LatLng>[]; final _ctrl = MapController(); // En la pantalla: StreamBuilder<Position>( stream: _gpsStream, builder: (ctx, snap) { if (snap.hasData) { final p = snap.data!; final ll = LatLng( p.latitude, p.longitude); // Añadir a la trayectoria _trayectoria.add(ll); // Mover cámara al usuario _ctrl.move(ll, _ctrl.camera.zoom); } return FlutterMap( mapController: _ctrl, children: [ TileLayer(...), PolylineLayer(polylines: [ Polyline( points: _trayectoria, color: Color(0xFF006A6A), strokeWidth: 4, ), ]), MarkerLayer(...), // posición actual ], ); }, )
📐
PolylineLayer
Recibe una lista de Polyline, cada uno con su lista de LatLng. strokeWidth en píxeles lógicos, independiente del zoom.
⚠️
No usar setState() dentro del StreamBuilder — ya reconstruye automáticamente. Llamar setState() adicional duplica los renders.
Hack #2
Haversine en Dart puro — distancia acumulada
La fórmula en Dart — sin dependencias
import 'dart:math'; /// Distancia en metros entre dos coordenadas double haversine(LatLng a, LatLng b) { const R = 6371000.0; // Radio Tierra en m final dLat = _rad(b.latitude - a.latitude); final dLon = _rad(b.longitude - a.longitude); final sinLat = sin(dLat / 2); final sinLon = sin(dLon / 2); final x = sinLat * sinLat + cos(_rad(a.latitude)) * cos(_rad(b.latitude)) * sinLon * sinLon; return R * 2 * asin(sqrt(x)); } double _rad(double deg) => deg * pi / 180; // Distancia total de la trayectoria double distanciaTotal(List<LatLng> pts) { double total = 0; for (int i = 1; i < pts.length; i++) { total += haversine(pts[i-1], pts[i]); } return total; }
✓
Verificación
haversine(LatLng(-16.50,-68.15), LatLng(-17.78,-63.18)) da ~547 km — exactamente la distancia La Paz → Santa Cruz.

Resumen
Lo que aprendiste hoy
  • Future = un valor, Stream = valores continuos — geolocator ofrece los dos
  • distanceFilter: 5 reduce las actualizaciones GPS al 20% — ahorra batería en campo
  • WidgetsBindingObserver pausa el stream GPS cuando la app va al background — sin código nativo
  • PolylineLayer dibuja la trayectoria como una línea continua que crece con cada punto GPS
  • Haversine en Dart puro — 10 líneas, sin dependencias, error menor al 0.5%
  • StreamBuilder reconstruye la UI automáticamente — no usar setState() dentro de él
