⚠️
Sin permisos el GPS no funciona
Si falta cualquiera de los dos permisos en el Manifest, geolocator lanza una excepción silenciosa y nunca emite posiciones.
✓
flutter pub get
Después de editar pubspec.yaml, ejecutar siempre flutter pub get antes de correr la app.
Concepto clave
Future vs Stream — la diferencia esencial
📸
Future — una foto
Devuelve un solo valor y termina. getCurrentPosition() da la posición actual una sola vez.

Úsalo para: posición inicial al abrir la app.
🎥
Stream — una película
Emite valores continuamente. getPositionStream() emite una nueva posición cada vez que el usuario se mueve.

Úsalo para: seguimiento GPS en tiempo real.
Los dos métodos — listo para copiar
// Future: posición actual (una vez) final pos = await Geolocator.getCurrentPosition( desiredAccuracy: LocationAccuracy.high); // Stream: posición continua final stream = Geolocator.getPositionStream( locationSettings: LocationSettings( accuracy: LocationAccuracy.high, distanceFilter: 5, ), );
💡
distanceFilter: 5 — el Stream solo emite cuando el usuario se mueve 5 metros. Ahorra hasta 80% de batería.

import 'package:geolocator/geolocator.dart'; import 'package:latlong2/latlong2.dart'; import 'dart:async'; class GpsService { StreamSubscription<Position>? _sub; final _ctrl = StreamController<LatLng>.broadcast(); Stream<LatLng> get posicion => _ctrl.stream; Future<void> iniciar() async { // 1. Verificar permisos var perm = await Geolocator.checkPermission(); if (perm == LocationPermission.denied) { perm = await Geolocator.requestPermission(); } if (perm == LocationPermission.deniedForever) return; // 2. Iniciar stream GPS _sub = Geolocator.getPositionStream( locationSettings: LocationSettings( accuracy: LocationAccuracy.high, distanceFilter: 5, ), ).listen((p) { _ctrl.add(LatLng( p.latitude, p.longitude)); }); } }
✓
StreamController.broadcast()
Permite que múltiples widgets escuchen el mismo GPS simultáneamente — el mapa, el contador de distancia y el formulario reciben la misma posición.
💡
Arquitectura 3 capas
GpsService vive en lib/services/. No importa widgets. La pantalla lo instancia y escucha el stream — separación limpia.

// Continúa la clase GpsService... // Pausa cuando la app va al fondo void pausar() => _sub?.pause(); // Reanuda cuando la app vuelve void reanudar() => _sub?.resume(); // Liberar recursos al cerrar void detener() { _sub?.cancel(); _ctrl.close(); } } // En MapScreen — usar el servicio: class _MapaState extends State<MapaScreen> with WidgetsBindingObserver { final _gps = GpsService(); @override void initState() { super.initState(); WidgetsBinding.instance .addObserver(this); _gps.iniciar(); } @override void didChangeAppLifecycleState( AppLifecycleState s) { if (s == AppLifecycleState.paused) _gps.pausar(); else if (s == AppLifecycleState.resumed) _gps.reanudar(); } @override void dispose() { _gps.detener(); WidgetsBinding.instance .removeObserver(this); super.dispose(); } }

final _trayectoria = <LatLng>[]; final _ctrl = MapController(); @override Widget build(BuildContext ctx) { return StreamBuilder<LatLng>( stream: _gps.posicion, builder: (ctx, snap) { if (snap.hasData) { _trayectoria.add(snap.data!); _ctrl.move( snap.data!, _ctrl.camera.zoom); } return FlutterMap( mapController: _ctrl, options: MapOptions( initialCenter: LatLng(-16.50, -68.15), initialZoom: 15, ), children: [ TileLayer(urlTemplate: 'https://tile.openstreetmap' '.org/{z}/{x}/{y}.png', userAgentPackageName: 'com.arteclab.geocollect'), PolylineLayer(polylines: [ Polyline( points: _trayectoria, color: Color(0xFF006A6A), strokeWidth: 4)]), MarkerLayer(markers: [ if (snap.hasData) _markerPosicion(snap.data!), ]), ], ); }, ); }

Distancia acumulada
Haversine en Dart puro — sin dependencias
lib/services/gps_service.dart — añadir
import 'dart:math'; /// Distancia en metros entre 2 puntos double haversine(LatLng a, LatLng b) { const R = 6371000.0; final dLat = _rad(b.latitude - a.latitude); final dLon = _rad(b.longitude - a.longitude); final h = pow(sin(dLat/2), 2) + cos(_rad(a.latitude)) * cos(_rad(b.latitude)) * pow(sin(dLon/2), 2); return R * 2 * asin(sqrt(h)); } double _rad(double d) => d * pi / 180; /// Distancia total de la trayectoria double distanciaTotal( List<LatLng> pts) { double total = 0; for (var i = 1; i < pts.length; i++) total += haversine(pts[i-1], pts[i]); return total; }
✓
Verificación rápida
haversine(LatLng(-16.50,-68.15), LatLng(-17.78,-63.18)) debe dar ~547 km — La Paz → Santa Cruz.

Resumen
Lo que aprendiste hoy
  • geolocator requiere declaración en pubspec.yaml y los dos permisos de ubicación en AndroidManifest.xml
  • Future = un valor · Stream = valores continuos. GPS en tiempo real es siempre un Stream
  • distanceFilter: 5 — el stream solo emite al moverse 5 metros, ahorra 80% de batería
  • WidgetsBindingObserver pausa el GPS al fondo y lo reanuda al volver — sin código nativo
  • PolylineLayer dibuja la trayectoria como línea continua sobre el mapa
  • Haversine en Dart puro — 12 líneas, sin dependencias, error menor al 0.5%
