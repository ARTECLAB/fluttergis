Formatos de exportacion
GeoJSON para GIS, CSV para todos los demas
🗺️
GeoJSON
Formato estandar para datos vectoriales. Lo abre QGIS, GeoServer, Leaflet, OpenLayers directamente. Incluye geometria + atributos.
📊
CSV
Lo abre Excel, Google Sheets, cualquier responsable de proyecto. lat/lon como columnas, atributos como el resto. Para el publico no-GIS.
⚠️
El error mas frecuente
GeoJSON usa [longitud, latitud] en coordinates. Poner [latitud, longitud] coloca los puntos en el oceano. Siempre: coordinates: [lon, lat].
✓
Generar GeoJSON en Dart puro (sin libreria) te hace independiente de dependencias externas y te enseña la estructura real del estandar.

Convertir List<Colecta> a FeatureCollection
lib/export/geojson_exporter.dart
String colectasAGeoJson( List<Colecta> colectas) { final features = colectas.map((c) => { 'type': 'Feature', 'geometry': { 'type': 'Point', // GeoJSON: [lon, lat] NO [lat, lon] 'coordinates': [c.longitud, c.latitud], }, 'properties': { 'id': c.id, 'tipo': c.tipo, 'descripcion': c.descripcion, 'timestamp': c.timestamp, 'foto': c.fotoPath != null, }, }).toList(); final fc = { 'type': 'FeatureCollection', 'features': features, }; return jsonEncode(fc); } // Guardar y compartir: Future<void> exportarGeoJson( List<Colecta> colectas) async { final json = colectasAGeoJson(colectas); final dir = await getApplicationDocumentsDirectory(); final file = File('${dir.path}/colectas.geojson'); await file.writeAsString(json); await Share.shareXFiles( [XFile(file.path)], text: 'Colectas GeoCollect — ARTECLAB', ); }

import 'dart:convert'; // HACK: link de preview en geojson.io // sin necesitar servidor propio String generarLinkPreview( List<Colecta> colectas) { final json = colectasAGeoJson(colectas); final b64 = base64Encode( utf8.encode(json)); return 'https://geojson.io' '/#data=data:application/json' ';base64,$b64'; } // Compartir el link por WhatsApp // El receptor ve los puntos en el mapa // sin instalar nada ni tener un servidor // Publicar directamente en GeoServer Future<void> publicarEnGeoServer( List<Colecta> colectas) async { final json = colectasAGeoJson(colectas); final resp = await http.post( Uri.parse('http://servidor:8080' '/geoserver/rest/workspaces' '/geocollect/datastores' '/colectas/featuretypes'), headers: { 'Content-Type': 'application/json', 'Authorization': 'Basic ' + base64Encode( utf8.encode('admin:geoserver')), }, body: json, ); if (resp.statusCode == 201) print('Publicado en GeoServer!'); }
✓
geojson.io via base64
El link codifica todo el GeoJSON en la URL. El receptor lo abre en el navegador y ve los puntos en el mapa inmediatamente. No necesita software GIS.
🌐
Publicar en GeoServer via REST desde la app cierra el ciclo: la app colecta datos en campo y los publica directamente como capa WMS — sin pasar por QGIS ni el escritorio.
Resumen
Lo que aprendiste hoy
  • GeoJSON en Dart puro: FeatureCollection con features de tipo Point — coordinates: [lon, lat]
  • lon antes que lat en GeoJSON — siempre al reves que LatLng(lat, lon) de flutter_map
  • share_plus.ShareXFiles comparte archivos con WhatsApp, email, Drive con una sola linea
  • Link geojson.io con base64: preview inmediato compartible sin servidor — funciona hasta ~500KB
  • REST API de GeoServer: POST con GeoJSON publica la capa directamente desde la app
  • CSV + GeoJSON: cubrir el publico GIS (QGIS) y el publico general (Excel, Sheets)
